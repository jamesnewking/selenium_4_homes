# Technologies: JS/Selenium v4/Jest/Page Object Model

## To run test with Jest

```
npm run test
```

## To configure Jest

https://jestjs.io/docs/en/configuration


>defined in `jest.config.json`

## Jest notes

to run only a few tests
```
test.only
describe.only

```

to skip tests

```
test.skip
describe.skip
```
